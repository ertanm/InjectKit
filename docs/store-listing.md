## Full Description

### PromptVault — Your AI Prompt Library

Stop re-typing the same prompts. PromptVault lets you save your best AI prompts and inject them into any AI chat from the extension popup.

**How it works:**
1. Save your frequently-used prompts organized by spaces and tags
2. Press **⌘K** (Mac) or **Ctrl+K** (Windows/Linux), or open the extension icon, on a supported AI chat
3. Search, select, and inject — or copy — in a few seconds

**Supported AI tools:**
- ChatGPT (chat.openai.com, chatgpt.com)
- Claude (claude.ai)
- v0 by Vercel (v0.dev, v0.app)

**Features:**
- Fast client-side search across your prompts
- Tag-based organization with filterable tabs
- One-click copy or inject into the active chat
- Cloud sync across devices when signed in
- Offline access from local cache when not connected
- Dark mode (follows system theme)
- Keyboard-first navigation in the popup (arrow keys + Enter)

**Pro (subscription, via Stripe):**
- Unlimited prompts (Free: 50)
- Additional Pro capabilities (version history, analytics, import/export) are available on the backend; the extension UI focuses on core library and inject flows — see the website or in-app messaging for the latest

**Privacy:**
- No ad tracking cookies
- We never read your AI conversations
- Privacy policy: use your deployed API host, e.g. `https://YOUR_API_HOST/privacy`

## Chrome Web Store — Privacy & permissions (summary)

- **Single purpose:** Save and inject user-authored prompts on supported AI chat sites.
- **storage:** Account session and local prompt cache.
- **scripting:** Inject text into the chat input on sites you use with PromptVault.
- **host_permissions:** Only the supported AI origins and your PromptVault API URL (for sync).
